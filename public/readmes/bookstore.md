# Bookstore Application

A full-stack e-commerce platform built with the MERN stack (MongoDB, Express.js, React.js, and Node.js).

### Customer Features
- **User Authentication**: Secure signup and login using JWT.
- **Book Catalog**: Browse a wide range of books with search and filter capabilities.
- **Category Filtering**: View books by specific categories.
- **Shopping Cart**: Add/remove books, update quantities.
- **Order Management**: Place orders and view order history.
- **Responsive Design**: Optimized for mobile, tablet, and desktop.

### Admin Features
- **Dashboard**: Overview of sales and inventory.
- **Inventory Management**: Add, update, and delete books.
- **Order Tracking**: Manage customer orders and status.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/kamalnathmurugan/bookstore.git
   ```

   ```bash
   cd server
   npm install
   npm start
   ```

3. **Frontend Setup**
   ```bash
   cd client
   npm install
   npm run dev
   ```
